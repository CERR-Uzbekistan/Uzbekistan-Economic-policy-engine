Cross-checks the JS impulse-response solver in qpm_uzbekistan/index.html against the canonical IRIS Toolbox implementation in data/Uzbekistan.model / data/readmodel_uzb.m. Benchmark IRFs come from a07_validate_vs_html.m in the CERR MATLAB pack (see data/matlab_irf_benchmark.csv).

Methodology

  • Both engines run in pure deviation form — pre-shock state = 0; all trends set to steady state.
  • Shock applied at Q0 with size 1 pp; horizon 12 quarters; same as a07_validate_vs_html.m.
  • Parameter set: HTML defaults from the slider DOM (P()) — these are intentionally calibrated to match readmodel_uzb.m point-for-point.
  • Phillips-curve note: the HTML model has an extra direct import-price term a₄·Δs that is not present in the MATLAB model. Tick the "a₄ = 0" box to make the equations exactly comparable. Untick to see the effect of the direct-pass-through extension.
  • Tolerance bands: pass < 0.05 pp max abs delta · warn < 0.15 pp · fail ≥ 0.15 pp.
